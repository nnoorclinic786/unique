# **App Name**: WholesaleRx

## Core Features:

- Admin Dashboard: Centralized dashboard for managing drugs, buyers, and overall system settings.
- Buyer Registration: Enable retail chemists, doctors, hospitals, and clinics to register as buyers, including collecting necessary documentation such as drugs registration, registered practitioner registration, GST number, and delivery address.
- Admin Drug Management: Allow admins to register new drugs, including details required for wholesale distribution.
- Order Placement: Buyers can place orders for drugs with cash on delivery as the primary payment mode.
- Order Management: Admin interface to manage incoming orders, track delivery, and update order status.
- Delivery Management: Provide functionality to manage the delivery process.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6), evokes trust and health.
- Background color: Soft, desaturated blue (#E3F2FD) creates a clean, professional feel.
- Accent color: Muted teal (#4DB6AC) for key actions, to indicate health and wellness.
- Headline font: 'Poppins' (sans-serif) for a modern and approachable feel, suitable for titles.
- Body font: 'PT Sans' (sans-serif) to ensure readability, to display essential information in an approachable way.
- Use clear, standard icons to represent drug categories and actions.
- Clean and organized layout to easily browse and manage drugs. Important details like drug name, price, and availability should be immediately visible.
